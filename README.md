## 🎵 Sobre el proyecto

**ONA Studio** es la plataforma web oficial de uno de los estudios de grabación más importantes de España. Un espacio donde han grabado **artistas de reconocimiento nacional** del panorama musical actual.

El proyecto consistió en el diseño y desarrollo completo del frontend: desde la arquitectura de componentes hasta la experiencia de usuario final, buscando transmitir la **esencia artística** del estudio a través de la web.

---

## ✨ Características de la plataforma

| Feature | Descripción |
|---|---|
| 🎨 **Diseño premium** | UI oscura y elegante que refleja la identidad del estudio |
| 🎵 **Reproductor integrado** | Escucha samples y producciones directamente en la web |
| 📸 **Galería inmersiva** | Fotografías de alta resolución de las salas y sesiones |
| 📅 **Reservas online** | Sistema de reserva de salas con calendario integrado |
| 🏆 **Artistas destacados** | Showcase de los artistas que han grabado en el estudio |
| 📱 **Mobile-first** | Experiencia optimizada para todos los dispositivos |
| ⚡ **Performance** | Carga ultrarrápida con lazy loading y optimización de assets |

---

## 🏗️ Arquitectura Frontend

```
onastudio/
├── src/
│   ├── components/
│   │   ├── Hero/          → Sección principal con vídeo de fondo
│   │   ├── Artists/       → Grid de artistas con hover effects
│   │   ├── Studios/       → Tour virtual por las salas
│   │   ├── Player/        → Reproductor de audio integrado
│   │   ├── Booking/       → Formulario de reservas
│   │   └── Gallery/       → Galería con lightbox
│   ├── hooks/             → Custom hooks reutilizables
│   ├── services/          → Llamadas a API
│   ├── styles/            → Design system y variables CSS
│   └── utils/             → Funciones auxiliares
├── public/
└── package.json
```

---

## 🎨 Decisiones de Diseño

- **Paleta oscura** con acentos en morado y oro — transmite exclusividad
- **Tipografía** cuidada para legibilidad y estética artística
- **Micro-animaciones** en hover y transiciones para sensación premium
- **Vídeo de fondo** en hero con overlay gradiente para texto legible
- **Layout asimétrico** en secciones para dinamismo visual
